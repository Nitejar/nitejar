Nitejar is built for operators who want speed without guesswork. Your fleet can triage, draft, sync, and alert across channels. You can always inspect what happened in the activity timeline, execution logs, and cost breakdowns.

## Core flow

1. Configure agents and skills for each job.
2. Connect plugins for Telegram, GitHub, Slack, or webhooks.
3. Use **Command Center** for what needs attention right now.
4. Use **Company** for portfolio state, staffing depth, and team load.
5. Use **Work** for goals, tickets, and queue movement.
6. Verify spend and limits in **Costs**.
7. Drop into **Activity** when you need the receipts.
