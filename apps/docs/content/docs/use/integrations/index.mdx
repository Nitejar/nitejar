Integrations are how Nitejar receives events and sends responses.

## Available setup guides

- **[Telegram Setup](/use/integrations/telegram-setup)**
- **[GitHub Setup](/use/integrations/github-setup)**

Choose a channel, assign agents, and verify delivery in Activity.

<Callout type="info" title="Where to verify">
Open **Admin > Plugins** for instance configuration and webhook health. Open **Admin > Activity** for run timelines triggered by each integration.
</Callout>
